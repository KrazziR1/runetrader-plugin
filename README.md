# RuneTrader GE Sync

A RuneLite plugin that syncs your Grand Exchange offers to [RuneTrader.gg](https://www.runetrader.gg) in real time.

## Features

- **Live GE slot tracking** — see all 8 of your GE slots update in real time on the web dashboard
- **Automatic flip detection** — buy and sell offers are matched automatically to track your profit, ROI, and tax
- **Flip history** — every completed flip is logged with buy price, sell price, quantity, tax, and profit
- **Merchant Mode** — full session analytics including GP/hr, capital deployed, daily P&L, and more
- **No manual entry required** — everything is tracked automatically from your in-game GE activity

## Setup

1. Create a free account at [runetrader.gg](https://www.runetrader.gg)
2. Go to **Settings** and generate an API key
3. In RuneLite, open the **RuneTrader GE Sync** plugin config
4. Paste your API key into the **API Key** field
5. Make sure **Enable Sync** is checked
6. Open your Grand Exchange in-game — your offers will start syncing immediately

## Privacy

This plugin only transmits your Grand Exchange offer data (item name, price, quantity, slot, status) to RuneTrader.gg. No other game data is collected. Your API key is stored locally in RuneLite's config and is never logged or shared.
